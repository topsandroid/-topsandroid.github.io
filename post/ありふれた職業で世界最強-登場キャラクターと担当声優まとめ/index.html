『ありふれた職業で世界最強』主要キャラクター

南雲ハジメ(なぐもはじめ)

声優深町寿成さん
天職錬成師
注目天職が非戦闘系のため周りからは無能とされていた。奈落に落ちた後は凄惨な道を辿り、元クラスメイトであっても敵であれば殺すという考えに至る。自他共に認めるオタクで、作り出した武器等に片鱗が見られる。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

ユエ

声優桑原由気さん
天職神子
注目奈落に封印されていた少女でハジメによって解放される。外見年齢は若いが吸血鬼であるため実年齢は300歳越え。度々人前でハジメといちゃついている。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

シア・ハウリア

声優髙橋ミナミさん
天職占術師
注目スタイル抜群の亜人。“未来視”の能力を持っている。故郷を追われた過去を持ち、帝国軍に追われていたところをハジメに救われた。ハジメ以外の男性には興味を示さない様子。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

ティオ・クラルス

声優日笠陽子さん
天職守護者
注目竜人族の女性でハジメとは竜化していた時に出会う。その時のとある出来事によりドＭに目覚めてしまった。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

白崎香織(しらさきかおり)

声優大西沙織さん
天職治癒師
注目ハジメのクラスメイトの少女で、地球にいた頃からハジメのことを気にかけていた。しかし香織はクラスのマドンナ的存在であったため、ハジメをかまうほど彼は孤立することになってしまう。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

八重樫雫(やえがししずく)

声優花守ゆみりさん
天職剣士
注目香織の親友。剣術に長けている。女性人気が高く、地球でも異世界でもファンが多くいる模様。登場人物たちの中では常識人な部類で、クラスの中でも比較的ハジメとは良好な関係を築いている。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

ミュウ

声優小倉唯さん
種族海人族
注目人攫いに監禁されていたところをハジメたちに救われた少女。その経緯からハジメのことを「パパ」と呼び慕っており、ハジメからも娘のように大切にされている。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

『ありふれた職業で世界最強』クラスメイト

畑山愛子(はたやまあいこ)

声優加隈亜衣さん
天職作農師
注目ハジメたちの学校の社会科教師。心優しい性格で異世界でも生徒たちのことを第一に考えている。教師・大人としての責任も持ち合わせており、ハジメを説教したことも。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

天之河光輝(あまのがわこうき)

声優柿原徹也さん
天職勇者パーティー光輝パーティー
注目容姿端麗、成績優秀、スポーツ万能の完璧超人。正義感が強い故に思い込みが強く、周囲の言葉を流してしまうこともしばしば。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

坂上龍太郎(さかがみりゅうたろう)

声優木島隆一さん
天職拳士パーティー光輝パーティー
注目光輝の親友かつ相棒。友達想いであるものの、デリカシーに掛けるところがある。豪快な性格で短慮な行動も目立つ。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

檜山大介(ひやまだいすけ)

声優白石稔さん
天職軽戦士パーティー小悪党組
注目香織に好意を抱いているため、香織から気にかけてもらっていたハジメのことを異世界に来る前から快く思っていなかった。ハジメが奈落に落ちた原因を作ったのも大介である。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

中村恵里(なかむらえり)

声優西明日香さん
天職降霊術師パーティー光輝パーティー
注目温和でおとなしい性格をしており、暴走しがちな鈴のストッパー。しかし優しい性格は表向きの顔で、本来の性格は利己的で残忍。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

谷口鈴(たにぐちすず)

声優伊藤彩沙さん
天職結界師パーティー光輝パーティー
注目明るくクラスのムードメーカー的な存在。美少女が好きらしく、よく香織にセクハラまがいなことをしている。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

園部優花(そのべゆうか)

声優富田美憂さん
天職投術師パーティー愛ちゃん護衛隊
注目愛ちゃん護衛隊のリーダー格。見た目は少し不良寄りでも性格はいたって真面目。ハジメのことが異性として気になる様子。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

玉井淳史(たまいあつし)

声優金子誠さん
天職曲刀師パーティー愛ちゃん護衛隊
注目愛ちゃん護衛隊のサブリーダー的存在。当初は自分たちの無力さに自虐的になっていた。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

宮崎奈々(みやざきなな)

声優紡木吏佐さん
天職氷術師パーティー愛ちゃん護衛隊
注目愛ちゃん護衛隊のムードメーカー。スレンダーなスタイルをしており、本人は気にしている様子。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

菅原妙子(すがわらたえこ)

声優峯田茉優さん
天職操鞭師パーティー愛ちゃん護衛隊
注目おっとりとした見た目をしているが、実はちょっぴりSなところがあるらしい。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

清水幸利(しみずゆきとし)

声優石田彰さん
天職闇術師パーティー愛ちゃん護衛隊
注目おとなしい性格をしている少年。ハジメと同じくオタクだが周りには隠していた。自己顕示欲がとても強く、それは異世界に来てから顕著に表れるようになった。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

遠藤浩介(えんどうこうすけ)

声優榎木淳弥さん
天職暗殺者パーティー永山パーティー
注目地球でも異世界でもとても影の薄い少年。その影の薄さは敵モンスターにもスルーされるほど。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

『ありふれた職業で世界最強』解放者

オスカー・オルクス

声優寺島拓篤さん
天職錬成師
注目オルクス大迷宮の主で外伝『零』では主人公を務める。メガネやメイド服には並みならぬこだわりがある。

© Ryo Shirakome, OVERLAP/ARIFURETA Project

ミレディ・ライセン

声優ゆかなさん
天職–
注目ライセン大迷宮の主で、魂をゴーレムに移すことで生きながらえている。公式サイトの紹介文にも書かれてしまうくらい“とにかくウザい”。

© Ryo Shirakome, OVERLAP/ARIFURETA Project
