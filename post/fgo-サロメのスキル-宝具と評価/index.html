サーヴァント名のスキル解説・強化優先度

宝具「あなたにくちづけしたわ」

効果
敵単体に超強力な攻撃
&呪いを付与(5T/1000)
&呪厄を付与(5T)

ヒット数が特に多いわけでもない普通の単体アーツ宝具で、低レアかつ強化なども行われていないということもあって、主力のアタッカーの宝具としてはそこそこの性能。とはいえ、サロメの宝具の真骨頂は通常運用ではなく、追加効果の呪厄の倍率が非常に高いこと。

スリップダメージを軸にした編成なら、低レアゆえのコストの低さも相まって唯一無二の活躍を見せてくれる。ただし、呪厄ダメージこそ高いものの、呪い自体のダメージはたったの1000なので、呪いの基礎ダメージが高いサーヴァントと組み合わせないとダメージを伸ばしにくいので注意が必要。

スキル1:血塗られた銀盆 B+ (CT7→5)

効果
自身に秩序かつ善特攻を付与(3T)
&攻撃力アップ(3T)
強化優先度
C
(後回しでも良い)

自身に攻撃バフと”秩序かつ善”特攻を付与するバフスキル。”秩序かつ善”特攻は「秩序であり、善であるサーヴァント」にしか効果がないので、秩序のみのサーヴァントや善のみのサーヴァントには通用しない。

攻撃バフ自体は無難なバフスキルなので、サロメを使いたい人は強化しておこう。

スキル2:天性の肉体(体) C (CT7→5)

効果
自身に弱体無効を付与(1回/3T)
&防御力アップ(3T)
&無敵を付与(1回/3T)
強化優先度
C
(後回しでも良い)

各種防御強化を付与できるスキル。防御強化解除で打ち消されることもありうるが、スキル3を軸に戦うなら頼りになる効果ではあるので、余裕があれば強化しておこう。CT短縮の恩恵が大きいため、レベル6、10を目指すのがおすすめ。

スキル3:七つのヴェールの踊り A (CT9→7)

効果
自身に毎ターンHP回復を付与(7T)
&「毎ターン自身の防御強化解除(1つ)」を付与(7T)<強化扱い>
&「7ターン後に味方全体のNPをチャージ」を付与(1回)
強化優先度
B
(サロメが好きなら絶対に強化しよう)

サロメ最大の特徴とも言える、7ターン後に最大100%のNPを味方全体に配布するスキル。毎ターン自身の防御強化が解除されるというデメリットこそあるものの、7ターンサロメを生き残らせることができれば大量のNPを獲得できる。

ちなみに強化解除されるとそれまでの蓄積が無意味になるので、強化解除を使う相手が天敵といえる。

アペンドスキル解説

優先度
>>

周回で使われるタイプではないので、優先したいスキルは特にないが、スリップ運用なら魔力装填は用意しておきたい。残るスキルはあまり影響がないので、余裕ができてから開放&強化するのがおすすめ。

ちなみにスリップ運用の場合、クリ発生耐性が邪魔になることもあるので、基本的には開放しないでおくのがおすすめ。

サロメの強い点、使える場面

①呪い&呪厄でスリップ大ダメージ

高倍率の呪厄宝具を持っているため、一部クエストで登場する超高倍率の特殊耐性やダメージカットを持つエネミーに対して効率よくダメージを与えられる。このタイプのクエストはめったに登場しないので、対策する必要自体あまりないが、簡単に入手できる低レアで対策できるサロメはもしもの場面で頼りになる存在と言える。

スリップ運用ならサロメのスキルはほとんど使われないので、最低限の育成で即戦力になるのも嬉しい(なんなら再臨する必要すらない)。

スリップの中でも編成難易度が低い(普及している鯖が軸になる)

呪い系スリップで強力なサーヴァントは蘆屋道満やモルガンなど、汎用性が高いサーヴァントが多い印象。やけどは楊貴妃や水着巴御前、毒は酒呑童子がパーティの核になるので、モルガンらと比較すると少しハードルが高い。

とりあえずスリップパーティが必要な場面になったら、とりあえず呪い軸の編成を試してみるのがおすすめ。

②7ターン待機してNP大量獲得

7ターン後に味方全体のNPを最大100%増加させるスキルを活かす運用もサロメらしい戦い方。とにかく敵の攻撃から毎ターン防御強化が解除されるバーサーカーを守り続ける戦いになるが、サロメはバーサーカーなので、宝具だけでなく通常攻撃によるクリティカルが出るだけでも致命傷になる恐れがある。そもそも攻撃を受けないのが最も効率の良い耐え方なので、タゲ集中スキルやタゲ集中礼装で敵の攻撃からサロメを守ろう。

③Wサロメで7ターン後の爆発力をアップ

サロメ自身は耐久に貢献できず、7ターン耐えることだけに集中するため、パーティがサロメ専用の編成になりがち。専用編成ならサロメが1騎でも2騎でも大きな差はないので、Wサロメにして7ターン後の爆発力を上げる編成がおすすめ。

ちなみに2騎目のサロメがスキル3を使うタイミングは1ターンズラす必要があるので、気をつけよう。同時に使うとNPが一気に200%チャージされてしまう。

サロメの弱い点、使えない場面

再現度は高いが性能は微妙

戯曲「サロメ」を再現したスキル3は非常によくできているが、FGOのゲームシステムとは噛み合っていないため、専用パーティを組まない限り、NP配布を見ることすら難しい。スキル3抜きのサロメはスリップ以外特徴のないの普通の低レアバーサーカーなので、普通に遊んでいる限りはサロメをわざわざ採用したい場面はほとんどないだろう。

運用自体は楽しいの好きな人はうまく使おう

規定ターンまで生き残るのが目的なので、やることが分かりやすく、最後の宝具チェイン連打もド派手なので、使っていて楽しいサーヴァントでもある。高難易度クエストでも勝ち筋が見つかることもあるので、サロメを使いたいならしっかり育成&ゲームシステムを理解して7ターン踊り切ることを目指せる編成を組み立てよう。
