1F：パラメータがチェックされる

第3カルパの1Fは、大まかに3箇所のエリアに分かれます。

それぞれのエリアには、主人公の特定パラメータ(マガタマ補正なし)によって通れたり、通れなかったり、別の場所にワープする扉がありますので、あらかじめマップを確認しておきましょう。

B1F：属性チェックの扉

第3カルパのB1Fには、主人公の属性に応じて開くかどうかが決まる扉が3箇所あります。扉の奥には闇ブローカーがおり、それぞれ異なる商品(悪魔)を売ってくれます。

属性チェックの扉は、1Fの3区画それぞれから繋がっていますので、希望の闇ブローカーの所に行く場合、パラメータ条件、属性条件両方を満たす必要があります。

追跡イベント

B2FとB3Fの2階層に渡って、ライドウが追いかけてくるイベントが発生します。

ライドウに捕まると戦闘が発生し、戦闘が終了するとスタート地点に戻されてしまいますので、ライドウをやり過ごして下の階層を目指しましょう。

B4Fでライドウ戦(2回目)が発生

ライドウとの追跡イベントを越え、B4Fまで到達すると、ライドウ戦(2回目)が発生します。基本的にはイケブクロの時と同じ技を使用するので、精神無効や物理耐性を用意しておきましょう。ライドウ戦の攻略情報
