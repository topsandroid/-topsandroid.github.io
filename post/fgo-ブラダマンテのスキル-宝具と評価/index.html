ブラダマンテのスキル解説・強化優先度

宝具「目映きは閃光の魔盾」

効果
自身の宝具威力アップ(1T)
+敵全体に強力な攻撃
&スタンを付与(1T/確率60%)
&クリ発生率ダウン(3T/20%)

全体クイック宝具でダメージ前の宝具バフで威力もそこそこ。宝具追加効果の確率スタンとクリ発生ダウンで、耐久に少しだけ貢献できるが、耐久には向かないQ主体のサーヴァントなので強力な追加効果とは言えないだろう。

威力やNP回収力、追加効果どこをとっても物足りない宝具なので、今後の強化に期待したい。

ブラダマンテにとっては貴重なバフスキルで、各カードのNP効率も上がるため強化優先度は高い。

スキル2:白羽の騎士 B+ (CT8→6)

効果
自身にガッツを付与(1回/3T)
&防御力アップ(3T)
強化優先度
C
(後回しでも良い)

ガッツ&防御バフという自身の耐久に貢献するスキル。自分のことしか守れない上、そもそもQ主体の耐久編成はあまり強力ではないので、ブラダマンテに耐久してもらうという運用自体が微妙。ブラダマンテが好きで、彼女をとにかく使いたいという人は強化しておこう。

&弱体解除
強化優先度
C
(後回しでも良い)

味方に付与できる弱体解除&NPチャージスキルで、NPチャージ量は最大20%。スキル強化で配布量が変わるが、そこまで劇的な変化ではないので、余裕があれば強化するという認識でOK。

アペンドスキル解説

優先度
>=

貴重なNPチャージスキルである魔力装填が優先。どこでも使いたいなら攻撃適正、とりあえずという場合は追撃技巧向上がおすすめだが、どうしても今すぐ強化したいという人以外は、魔力装填だけ開放&強化しておくのが無難。

ブラダマンテの強い点、使える場面

①弱体解除ができる

スキルで味方単体を弱体解除できる。おまけでNP配布もできるので、開幕デバフを使ってくる高難易度などに適正がある。

ブラダマンテの弱い点、使えない場面

スキル効果が全体的に控えめ

星5サーヴァントらしい超強力なスキルを所持していない。宝具強化をされていたり、ステータスが優秀というわけでもないので、他のサーヴァントを差し置いて強いと言える部分があまりないのが残念。
